Domain Definition of the GPDS Value Sets

The GDPS is a first step for interoperable CROM/PROM - solutions in Germany. The domains have been decided mostly on the basis of ICF categories within the Questionnaire mappings it was decided that other categories had to be introduced. As the ICF coding options are heavily determined by functionality these are not sufficient enough as categorizations for describing mental health in general.

Also, coding options for measuring emotionality functions are limited. Thus, we decided upon using the HiTOP hierarchy as another factor for assigning the coding possibilities to Value Sets.

In total, we categorized the codes in 25 Value Sets with different coding options.

Fig. Value Set Domains and Value Sets

On top of that, there are Value Sets describing the MADRS, Kidscreen-27, CDRS-R and AMDP and CASCAP-2, that can only be shown as Value Sets and not as FHIR Questionnaire Instances because of license issues.
